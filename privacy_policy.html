Privacy Policy

Politique de confidentialité

Dernière mise à jour : 3 novembre 2025

Votre vie privée est importante pour nous. Cette politique de confidentialité explique quelles informations nous collectons, comment nous les utilisons, et vos droits concernant vos données personnelles.

1. Informations que nous collectons

1.1 Informations TikTok

Important : Lorsque vous connectez votre compte TikTok à notre application, nous collectons et utilisons les données suivantes via l'API TikTok :
Type de données Description Utilisation
Informations de profil Nom d'utilisateur, photo de profil, biographie, ID utilisateur Affichage de votre profil dans l'application
Statistiques du compte Nombre de followers, de vues, de likes Affichage de vos statistiques et analyses
Liste de vidéos Métadonnées de vos vidéos TikTok Gestion de votre contenu vidéo
Certification artiste Statut de certification Vérification et affichage du statut
Token d'accès Jeton d'authentification TikTok Authentification et autorisation API

1.2 Informations que vous fournissez

  • Informations d'inscription (email, nom)
  • Contenu que vous créez ou téléchargez
  • Communications avec notre support

1.3 Informations collectées automatiquement

  • Adresse IP et informations de l'appareil
  • Type de navigateur et système d'exploitation
  • Pages visitées et durée d'utilisation
  • Cookies et technologies similaires

2. Comment nous utilisons vos informations

Nous utilisons vos données pour :

  • Fournir le Service : Vous permettre de publier et gérer du contenu sur TikTok
  • Authentification : Vérifier votre identité via TikTok Login Kit
  • Afficher vos statistiques : Présenter vos données de performance TikTok
  • Publication de contenu : Télécharger des vidéos sur votre compte TikTok via l'API
  • Améliorer le Service : Analyser l'utilisation pour optimiser l'expérience
  • Communication : Vous envoyer des notifications importantes concernant le Service
  • Conformité : Respecter les obligations légales et les conditions de TikTok

3. Intégration avec TikTok

Relation avec TikTok :

Notre application utilise l'API TikTok for Developers. En utilisant notre Service :

  • Vos données sont également soumises à la Politique de confidentialité de TikTok
  • Nous agissons comme intermédiaire entre vous et TikTok
  • TikTok peut collecter des données supplémentaires selon sa propre politique
  • Nous respectons les politiques développeur de TikTok

Autorisations TikTok (Scopes)

Notre application demande les autorisations suivantes :

  • user.info.basic : Accès aux informations de base de votre profil
  • user.info.profile : Accès aux détails de votre profil
  • user.info.stats : Accès à vos statistiques (followers, vues, etc.)
  • video.list : Lecture de la liste de vos vidéos
  • video.upload : Publication de vidéos sur votre compte
  • artist.certification.read : Lecture de votre statut de certification
  • artist.certification.update : Mise à jour de votre statut de certification

4. Partage de vos informations

Nous ne vendons jamais vos données personnelles. Nous pouvons partager vos informations avec :

  • TikTok : Pour publier du contenu et accéder aux fonctionnalités autorisées
  • Fournisseurs de services : Hébergement, analyse, support technique (sous NDA)
  • Conformité légale : Si requis par la loi ou pour protéger nos droits

5. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées :

  • Chiffrement SSL/TLS pour les transmissions de données
  • Stockage sécurisé des tokens d'accès TikTok
  • Accès restreint aux données personnelles
  • Surveillance régulière de la sécurité
  • Respect des meilleures pratiques de l'API TikTok

6. Conservation des données

Nous conservons vos données :

  • Aussi longtemps que votre compte est actif
  • Aussi longtemps que nécessaire pour fournir le Service
  • Pour respecter nos obligations légales

Les tokens d'accès TikTok sont conservés jusqu'à leur révocation ou expiration.

7. Vos droits

Vous avez le droit de :

  • Accès : Demander une copie de vos données personnelles
  • Rectification : Corriger des données inexactes
  • Suppression : Demander la suppression de vos données
  • Révocation : Révoquer l'accès à votre compte TikTok à tout moment
  • Portabilité : Recevoir vos données dans un format structuré
  • Opposition : Vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : privacy@votredomaine.com

8. Révocation de l'accès TikTok

Vous pouvez révoquer l'accès de notre application à votre compte TikTok à tout moment :

  • Via les paramètres de votre application
  • Via les paramètres de votre compte TikTok → Apps et sites web
  • En nous contactant directement

La révocation supprimera notre accès à vos données TikTok, mais peut limiter les fonctionnalités du Service.

9. Cookies et technologies de suivi

Nous utilisons des cookies pour :

  • Maintenir votre session connectée
  • Mémoriser vos préférences
  • Analyser l'utilisation du Service

Vous pouvez contrôler les cookies via les paramètres de votre navigateur.

10. Services tiers

Notre Service peut contenir des liens vers des sites tiers. Nous ne sommes pas responsables de leurs pratiques de confidentialité. Nous vous encourageons à lire leurs politiques de confidentialité.

11. Utilisateurs mineurs

Notre Service n'est pas destiné aux personnes de moins de 13 ans. Si vous êtes un parent et découvrez que votre enfant nous a fourni des données, contactez-nous immédiatement.

12. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications importantes seront notifiées par email ou via une notification dans l'application. La date de "dernière mise à jour" en haut de ce document indique quand la politique a été modifiée.

13. Conformité RGPD et CCPA

Pour les utilisateurs de l'UE (RGPD) et de Californie (CCPA), vous disposez de droits supplémentaires concernant vos données personnelles. Contactez-nous pour plus d'informations.
